
# 📊 Análisis de Embudo de Ventas

Este proyecto utiliza Python y Jupyter Notebook para analizar el comportamiento de los usuarios dentro de un embudo de ventas.  
El objetivo es identificar en qué etapas se producen mayores abandonos y cómo los cambios en la aplicación afectan la conversión final.

---

Requisitos
- Python 3.10+
- Jupyter Notebook
- Librerías:
  - pandas
  - matplotlib
  - seaborn

Instalación rápida:
```bash
pip install pandas matplotlib seaborn
